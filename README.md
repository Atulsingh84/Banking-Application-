Banking Application (Core Java - OOP Project)

A console-based Banking Application built using Core Java and Object-Oriented Programming principles.
This application simulates basic banking features such as account creation, deposit, withdrawal, 
Account Statements and List Of Account.

Features
    1) Open Account
    2) Deposit
    3) Withdraw
    4) Transfer
    5) Account Statement
    6) List Accounts
    7) Search Accounts by Customer Name
    0) Exit
 Concepts Used
  Classes & Objects
  Encapsulation
  Inheritance & Abstraction
  Polymorphism (method override)
  Exception Handling
  Collections Framework
  Layered Architecture (Repository, Service, Domain).
